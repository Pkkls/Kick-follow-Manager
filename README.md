
# Kick-follow-Manager

Export and re-follow all your Kick.com channels on a new account.

## 🌐 Project Page
👉 **[pkkls.github.io/Kick-follow-Manager](https://pkkls.github.io/Kick-follow-Manager/)**

## 📦 Files
- `kick_selenium.py` — Main script (undetected Chrome)
- `kick_follows.txt` — Your exported follows list

## 🚀 Quick Start
```bash
pip install undetected-chromedriver
python kick_selenium.py
```
Login via Google SSO or standard
## 📋 Requirements
- Python 3.8+
- Google Chrome installed
